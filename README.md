### About
This is a plugin for parkour mini game.

### Feature
* Parkour Game
* PlayDemo (Replay)
* BackBlocks (with edge blocker)
* Categories Menu (EASY, MEDIUM, HARD, EXTREME, EVENT, NO_CATEGORY, KZ, DROPPER)
* Stick with teleport behind maps
* Blocked McDown bots
* And more

### Support Version
1.16 - 1.17.1
